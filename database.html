字段 类型 备注

字段 类型 备注
pokemonName string 名称
property string 属性
hp int 血量（红突满特训满个体）
pa int 物攻（红突满特训满个体）
pd int 物防（红突满特训满个体）
sa int 特攻（红突满特训满个体）
sd int 特防（红突满特训满个体）
speed int 速度（红突满特训满个体）
specialTraining string 特训加成
skin string 皮肤加成
discount double 折扣（5.5/6.5/8/9/9.8/10）
exclusive Pointer 对应的专属数据
名称后缀加了专属的为已带专属，该对应精灵专属加的是种族值，非百分比
